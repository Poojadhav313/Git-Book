Create .git
>> git init

Add all files
>> git add .

Commit
>> git commit -m "msg"

Add remote url at the time of creating new repo
>> git remote add origin http://

Push changes
>> git pust -u origin master


----- Other stuff -----
View previous commit history
>> git log

Check remote location
>> git remote -v

Change remote location in case of changing repo name in github
>> git remote set-url origin http://newUrl

Undo last pushed commit without affection other commits
>> git reset HEAD~1
>> git push -f origin master
